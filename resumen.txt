----> INFERENCIA CON MEDIAS MUESTRALES <----
x- Prueba t de student
La prueba t de Student, basada en la distribucion t, es la alternativa mas ampliamente empleada para inferir acerca
de una o dos medias muestras. No opera bajo el supuesto de normalidad.
+- Las observaciones son independientes entre si.
+- Las observaciones provienen de una distribucion cercana a la normal.
	1.- Prueba t para una muestra
	2.- Prueba t para dos muestras pareadas
	3.- Prueba t para dos muestras independientes
		+- Cada muestra cumple las condiciones para usar la distribucion t.
		+- Las muestras son independientes entre si.
